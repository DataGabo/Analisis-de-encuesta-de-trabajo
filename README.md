# Analisis-de-encuesta-de-trabajo
El dashboard toma como base una encuesta de libre acceso que forma parte del curso de Power Bi de la cámara de comercio de Lima. A través de elementos visuales se grafica la distribución de hombres y mujeres entre los encuestados, su ocupación, estado civil, bar favorito en Lima, número de visitas al local, la ocasión y el motivo de la visita.
